扬州市宝应县莲藕汁——让莲藕汁走进年轻群体

一、 整体概述

1.1项目概述

中国荷藕之乡”—宝应县，地处江苏省中部，千年古运河穿镜而过，20多万亩的荷塘种植面积，气候土壤皆宜，处于南北交替之间，人称“鱼米之乡”，当地盛产荷（莲）藕。荷（莲）藕作为宝应县特产，更是成为了中国国家地理标志产品。由于盛产藕，同时该农作物在当地进行了深加工，除了生产藕片、藕粉，A公司更是将藕制作成饮品，即莲藕汁。此产品在老几代人群体中较为熟悉，但是现在，在年轻群体中并不为熟知，常年的莲藕滞销时刻困扰着包括宝应县在内众多依赖莲藕产业的县级市。因此，如何让更多年轻人认识到这款独具当地特色的健康饮品乃至莲藕其它副产品，将是本项目研究的方向。

1.2公司简介

A公司扎根当地近20年，经营范围包括方便食品（其他方便食品）、罐头（其他罐头）、饮料（果汁及蔬菜汁类、蛋白饮料类）加工、销售；预包装食品兼散装食品批发与零售。当然，旗下也销售莲藕汁。

1.3 政策背景

全国正在掀起乡村振兴的浪潮，即为了推动乡村产业高质量发展，培育乡村新产业新业态。继续支持创建农业产业强镇、现代农业产业园、优势特色产业集群。支持国家农村产业融合发展示范园建设。深入推进农业现代化示范区建设。实施文化产业赋能乡村振兴计划。实施乡村休闲旅游精品工程，推动乡村民宿提质升级。深入实施“数商兴农”和“互联网+”农产品出村进城工程，鼓励发展农产品电商直采、定制生产等模式，建设农副产品直播电商基地。

扬州市政府响应号召，为持续推进政府采购支持乡村振兴政策有效落实，助力巩固脱贫攻坚成果与乡村振兴有效衔接，市财政局积极贯彻落实财政部、省财政厅关于做好政府采购脱贫地区农副产品工作的指示精神，积极组织各县（市、区）财政局和市直各预算单位开展脱贫地区农副产品采购工作。

一是提高思想认识，深刻理解支持乡村产业振兴重要意义。

二是科学统筹谋划，及时部署脱贫地区农副产品采购工作。

三是强化督促指导，确保政府采购支持乡村振兴取得实效。

对于宝应县政府，2022年，该县被列入建立健全生态产品价值实现机制省级试点，为有力推动工作开展，加快试点建设，进一步提高金融机构对生态产品价值实现的支持力度，创新绿色金融产品和服务，有效解决生态产品抵押难、变现难问题。经研究，也出台了《关于加强宝应县生态产品价值实现金融支持的实施意见》。助推区域公用品牌建设，加强区域品牌的品质管控，规范生产、加工流程等标准化模式，县有关部门联合建立宝应荷藕、宝应大米等国家地理标志农产品和中国地理标志商标等区域品牌白名单，各银行机构加大对区域品牌白名单内生产主体的支持力度，提升对白名单内生产主体的授信额度，形成执行名单制管理，简化审批程序，提高审批效率。

宝应县政府还召开全县打造优势特色荷藕产业座谈会。县政府副县长王辉森，县农业农村局局长吴爱平、党委副书记、县农推中心主任杨明、副局长仲纪忙以及相关科室负责人、县政府办、县科技局、市场监管局、财政局、工信局、射阳湖镇、广洋湖镇、望直港镇、鲁垛镇等分管负责人、种植大户、营销企业及市级以上农业龙头企业（荷藕）主要负责人等30多人出席会议。

会议首先由生产经营主体畅谈宝应荷藕产业基础、发展愿望、制约瓶颈、对政府服务需求等，各代表畅所欲言，希望政府加强重视我县荷藕产业发展；代表呼吁广大企业要坚守行业自律，发展好荷藕行业协会协调自律作用，严惩恶性竞争，诚信经营，抱团发展，寻求共赢。接着，县政府办、县财政局、工信局、市场监管局、科技局等职能部门交流发言，表示将在政策范围内提供财政支持、项目支持、科技支撑、市场监管净化等。县农业农村局局长吴爱平表示，针对我县荷藕连片面积少、效益不稳定、产品单一等问题，今后将从技术研发、模式研究、市场品牌、组织推动、行业自律等方面开展相关工作。

最后，县政府副县长王辉森表示，荷藕行业需规范、整合，加强信息整合，充分发挥县荷藕行业协会的桥梁作用，及时与各职能部门沟通；企业间加强协作，相互之间走访、参观、学习，提升自身能力；对扶持政策进行调整，将新型专利、新型包装设计等考虑作为扶持重要方面；莲藕研究院成立后，加强与各职能部门合作、对接，各部门齐力助推研究院的发展；由县政府办牵头，尽快开展荷藕市场专项整治行动。各单位要构建起共建共治共享的新格局，加大执法力度，提升服务能力，完善基层管理，实现各部门协调联动，保证荷藕产业稳定有序发展，凝心聚力打造好我县“荷藕之乡”品牌。

基于以上背景，A公司将利用此次机会，实现自身产品的创新性发展。同时也能够，带动当地经济可持续发展，增加政府财政收入，提高当地农户的收入水平，并以此来助力乡村振兴发展。

1.4 当地背景

扬中，因其所处的长江下游段又称扬子江，取“扬子江中”之意而得名。成洲千年，建置百年，“鱼米之乡，江中明珠”便是它流芳全国的美誉之一。扬州就是坐落在江边的一座城市。而宝应则位于扬州境内，地处长三角和里下河地区，东接建湖、盐城、兴化，南连高邮，西与金湖、洪泽隔宝应湖、白马湖相望，北和淮安毗邻。

宝应县是中国荷藕之乡，地处江苏扬州北部，是国家级出口食品农产品质量安全示范区。丰富的自然资源加上良好的生态环境，使得宝应成为一个天然的聚宝盘，已逐步形成了种植、加工、外销的荷藕产业化链条，连续多年夺得荷藕种植面积、荷藕产量、荷藕出口量三项中国第一。

二、市场分析

2.1 市场规模与趋势

宝应县基于得天独厚的地理环境、水文地质条件和气候条件等，培育的荷藕具有颜色雪白、香气十足、淀粉量十分充足的优良品质。1950年，宝应开始逐步推广开展荷藕种植。1998年，宝应县被命名为“中国荷藕之乡”，宝应荷藕开始享誉全国，宝应荷藕的种植面积可达25万亩左右，年交易量50万多吨。2004年7月，“宝应荷藕”成功认证为国家农产品地理标志。2011年，该品牌位列中国农产品区域公用品牌百强。2017年“宝应荷藕”品牌价值为21.09亿元；2018年荷藕产加销休闲观光全产业链营业收入45.6亿元。2020年，宝应荷藕被认定为中国特色农产品优势区（第三批）。由此可见，当地对藕产业发展的重视，在不断的该产业的经济收入和地位。

2.2 适用人群

适合成长中的儿童、学习中的学生、亚健康的白领、轻断食的女士、锻炼中的年轻人和调养中的老年人。

2.3 竞争分析（优劣势分析）

莲藕汁市场中发展的优势主要包括以下几个方面：

1.市场规模大：莲藕种植在中医理疗和保健方面有广泛应用，全国市场庞大且前景可期。

2.消费趋势健康：随着人们生活水平的提高和对健康的关注度增加，健康食品的需求也

渐增加。

3.市场对象多样：原材料莲藕属于健康食品类，因此市场需要设法通过科学的市场分析、规划和策略，针对不同的消费群体进行定位和挖掘。

4.政策利好。各级政府积极推进莲藕产业的发展，并出台相关政策支持，原材料收购来源丰富。

5.产业潜力大：目前宝应县注重荷藕本身价值而对藕的副产品认知较为匮乏，且目前市场缺少相关品牌与公司占有荷藕副产品市场，因而有很大的市场潜力。

但目前藕汁市场发展仍有一定的劣势，主要体现为：

  1. 在年轻群体或部分地区中知名度较小
  2. 缺乏较为专业的电商平台进行售卖
  3. 藕汁产业链亟需完善
  4. 产品相关技术与原味口感仍有一些问题
  5. 用于制作莲藕汁的莲藕量偏少

为此，本公司希望利用电商各大平台，通过年轻人喜闻乐见的方式宣传，以提高该产品的竞争力和知名度。

三、产品介绍

3.1 产品描述

莲藕是生活中常见的蔬菜，可以根据不同的饮食需要制作成不同的菜式，也可以用来榨成莲藕汁饮用。莲藕汁味道鲜美，富含多种人体所需要的营养元素：

1.莲藕的热量和土豆相当，碳水化合物和脂肪的含量比较低，蛋白质的量较土豆稍高。

2.莲藕中含有维生素和微量元素，尤其是维生素K、维生素C，铁和钾的含量较高。

3.莲藕中含有黏液蛋白和膳食纤维，能与人体内胆酸盐，食物中的胆固醇及甘油三酯结合，使其从粪便中排出，从而减少脂类的吸收。

4.藕含有大量的单宁酸和丰富的植物纤维、维生素B12。

此外，具有健脾开胃、清热凉血、补益气血等功效：

1、健脾开胃：莲藕汁有一种与众不同的芳香，还带有鞣质，有一定健脾胃、缓解腹泻的功效，能提高胃口、促消化、健脾开胃，有利于胃纳不佳、食欲不佳者尽快恢复。

2、清热凉血：莲藕汁性寒，有清热凉血的功效，可用于感冒发烧等热证，莲藕味甘多液，对改善热症口干、衄血、咯血、下血等症状效果较好。

3、补益气血：莲藕汁的营养成分高，含有铁、钙等营养元素，绿色植物蛋白、维生素及木薯淀粉成分也较丰富，有较好的滋补血气的功效。

3.2 制作工艺

产品由莲藕汁专业生产厂家—A公司采用生榨工艺精制而成。最大程度地保持了莲藕中特有的营养成份，富含天然莲藕的独特风味。

四、营销策略

4.0 营销方式

本公司在注重产品质量的同时更加重视产品服务和受众反馈，打造亲民的产品形象。设计包装时会写上公司相关负责人的私人号码，进行真实的线上反馈。在饮品背后会设计有奖问卷二维码，尽最大可能获取用户对产品的真实评价，包括价格、口感、包装设计等一系列问题。

本公司还注重线上线下“双渗透”模式：线下从扬州市开始，渗透小卖部和大超市的供应商，以低利润为出发点迅速抢占饮品市场份额，在大超市中采用试喝、抽奖等模式吸引顾客；线上在知名电商平台开店铺的同时重视短视频渗透，用邀请博主测评新产品、植入软广的方式逐步渗透，配合较为魔性、接地气的广告词吸引顾客对产品的关注。

4.1 品牌定位

品牌定位通常品牌定位的核心是打造一系列具有高度辨识度和独特性的饮品，以建立品牌口碑和忠诚度。产品出现的目的是为了让每个平民百姓喝上健康快乐且好喝的饮料，我们不想让这款产品昙花一现，就要保证产品的质量能够过关。在饭桌上：可以选择大杯款；在日常饮用时，可以选择易拉罐款。除了常规包装，结合市场需求，还将推出自己设计的二次元包装，并开发全新莲藕果蔬混合口味，可满足众多年轻人的的情感需求和尝试新品的想法。在市场竞争激烈的情况下，不断推陈出新、引入新的元素和技术将有助于品牌的长期发展和用户的持续关注。

虽然莲藕汁饮品在市场有一定潜力，但是目前也存在众多风险，如不受市场青睐、口感不佳等问题。面对这些问题，我们极力解决，并利用网站和小程序便于获得反馈。在服务上，我们尽力提供全中国最好的服务，从售前的品尝到售后的服务，直接与产品经理交流，提出产品问题。为此，我们专门建立一个网站，主要有下列几个功能：

  1. 关于我们：从公司创立开始说起，简单介绍莲藕汁从无到有的经历，并介绍员工管理层和流水线，在展现莲藕汁生产的每一个步骤体现我们公司的企业文化。
  2. 产品介绍：重点介绍目前莲藕种植园的分布和专业化、现代化状况以及加工厂的分布情况，并列出当前产品的种类和销量情况。
  3. 产品选购：网站作为官方平台可以直接进行产品选购，目前主要是原味莲藕汁和复合果蔬莲藕汁。
  4. 莲藕知识：着重介绍莲藕汁对人体带来的一些好处，普及一些与莲藕相关的科学小知识；同时写明不适宜饮用莲藕汁的受众对象。
  5. 新闻报道：详细介绍当前公司的发展状况和市场的受欢迎程度，透明公开关键决策和部分财政收入。

6.论坛反馈：这是每一位用户能讨论与提出建议的平台，置顶是产品经理的私人电话和公司电话，产品有问题可以第一时间进行有效反馈。我们也会定期查看论坛讨论的一些问题进而帮助莲藕产品在未来的发展与改善进行一些重要调整，真正成为一款国民饮品。

4.2 消费人群

短期时间内，追求广撒网，以新颖产品为噱头吸引各年龄段的人群进行购买品尝，建立用户基础；长期时间追求保留先前用户基础，在产品的包装、口味与销售方式做出细微改动，创新产品，吸引特定群体，如二次元、健康生活人等。

4.3 独特卖点

  1. 高质量原料：莲藕都是从各地优质的专业种种植园精心挑选，保证莲藕优秀的生长环境。

2．产品绿色健康性：与可乐等知名饮料不同，莲藕汁是一款绿色饮品，生榨莲藕汁对身心健康有一定的作用，在享受饮料的味道同时还能尽量避免饮料对人体造成的一些负面影响。

3．抢先品尝版：在正式销售前退出抢先品尝版，进行小部分的市场调研，既能为之后正式售卖莲藕汁进行宣传，还能够进行小部分调整和改良产品。

  1. 创新的服务体验：我们坚信“顾客就是上帝”这句话，初心是在帮助农村脱贫致富的同时能够给顾客一些良好的体验，无论是产品设计，还是产品质量，或是产品售后服务，只要有问题，直接打电话给经理与董事长。

5.与潮流与二次元接轨：顺应时代发展，我们在保证产品质量的同时还注重包装设计和元素融合，设计属于莲藕自己的莲藕二次元形象莲藕娘。

总的来说，莲藕汁的独特卖点在于高质量原料、独特的设计风格、抢先款品尝、创新的服务体验以及独特的包装设计与元素融合。通过这些独特卖点，品牌能够吸引各年龄层顾客的购买欲望，推动产品的销售和品牌的发展。

五、财务计划

在政策大背景的支撑下，一定得投资是必不可少的。如果在当地政府的大力支持，并且利用好网络平台这个重要赛道，相信也是能实现弯道超车的，其回报远远不只局限于对企业本身带来资金，还对当地将带动经济发展和消费升级。此外，延长产业链的同时，也能实现灵活就业。

七、未来发展愿景及可行性分析

7.1其他发展方向

形成一定规模后，积累口碑，在保证产品本身质量的同时加强营销，增加受众群体，进而利用品牌效应售卖周边产品，例如杯身印有“荷”，“藕”等具象化的二次元虚拟形象的杯子，亦或是大力发展莲藕的其他相关产业，例如藕粉，对藕粉进行创新改良，增加多种配料，变成八宝粥类似的饮品。

7.2可行性分析

（一）经济可行性

目前，宝应县已获批“全国绿色食品原料（莲藕）标准化生产基地”，总面积达12万亩，总产量12万吨，共有基地单元8个、生产单位68个、基地农户3746户。基地联接县内加工企业8家，其中7家是绿色食品企业，对接面积9万亩，年产标准绿色食品莲藕原料8.5万吨，莲藕的绿色食品开发率为75%。全县建成集种植、加工、流通、观光、科研、文化“六位一体”的莲藕全产业链融合发展模式，拥有莲藕加工企业100多家，其中县级以上龙头企业29家，国家级农业产业化重点龙头企业1家（江苏荷仙食品集团）、省级4家、市级11家、县级13家，年加工藕制品20万吨左右，产量和模式均已成熟。

（二）市场可行性

自上世纪90年代以来，我国绿色食品迅速发展。绿色食品包括一系列加工、未经加工的食品，如农作物、牲畜和水产品。随着我国人均收入的和消费者对安全健康食品的意识持续快速增长，越来越多的人选择购买绿色食品。2017年前，就有10895家公司的25746件产品通过了绿色食品认证。仅2017年一年，我国新增绿色食品认证的企业就有4422家，新增绿色产品有10093件。而莲藕汁作为绿色饮品在饮料市场中是极度新奇与稀缺的，其口味也与一般饮料具有鲜明的差别，在如今同质化异常严重的饮料市场，各种“工业香精”不断推出，让消费者甚至只愿买瓶矿泉水，消费需求已从各种口味转变为绿色健康，由此衍生出的低糖低脂概念畅销全国。

2020年中国饮料市场规模突破1万亿元，预计2022年这一规模将达到12478.0亿元。随着中国经济的发展、居民消费水平的提升及消费结构的升级，中国饮料行业整体呈现出良好的增长态势。调研数据显示，中国消费者表示对碳酸饮料包装的偏好为易拉罐（74.2%），饮料品类方面，中国饮料消费者经常喝的饮料品类较多，主要为包装饮用水（62.7%），碳酸饮料（55.0%），奶制品（54.0%），气泡水（42.0%）等。由于饮料的主要成分是水，在大部分佐餐、出游的情况下，消费者选择饮品的主要目的是为了补充水分，且包装饮用水相较于其他饮品来说，添加剂较少，较为健康，因此在一定程度上消费者更偏爱包装饮用水。而莲藕汁较为符合当前市场环境，欠缺的只是宣传和口碑。

2、场地选择方面

宝应县荷藕产区内的土壤质地良好，疏松肥沃；酸碱适宜（PH值5.5～7），土层深30至40厘米，保水力强；有机质含量达2.5%至4%左右，适栽种植面积达25万亩。宝应县基于得天独厚的地理环境、水文地质条件和气候条件等，培育的荷藕具有颜色雪白、香气十足、淀粉量十分充足的优良品质。宝应县以射阳湖、广洋湖、望直港和鲁垛为主体规划建设了现代农业产业园，年总产值约59.76亿元。主导产业荷藕种植面积约10.8万亩，年产值达44.69亿元。

之后还会考虑在磁县、临沂等地区加强合作，建立质量更佳的莲藕种植园。
